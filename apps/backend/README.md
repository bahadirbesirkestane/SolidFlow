# Backend

Bu klasor uygulamanin backend tarafini icerir.

Ana sorumluluklar:

- HTTP API sunmak
- Solid tarama ve is akisi kurallarini calistirmak
- SQLite ve JSON veri katmanini yonetmek
- Excel, CSV ve PDF raporlari uretmek

Giris noktasi:

- `src/server.js`

Katmanlar:

- `src/domain`
- `src/application`
- `src/infrastructure`
- `src/presentation`
